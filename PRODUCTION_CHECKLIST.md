# AgentGuard 生产部署检查清单

## 🔒 安全检查

### 密钥和认证
- [ ] 所有默认密码已更改
- [ ] JWT 密钥至少 32 字符
- [ ] 数据库密码强度足够
- [ ] Redis 密码已设置
- [ ] API 密钥轮换策略已制定
- [ ] 私钥文件权限设为 600

### SSL/TLS
- [ ] 有效的 SSL 证书（非自签名）
- [ ] 强制 HTTPS 重定向
- [ ] HSTS 头已配置
- [ ] TLS 1.2+ 协议
- [ ] 安全的密码套件

### 网络安全
- [ ] 防火墙规则已配置
- [ ] 只开放必要端口（80, 443）
- [ ] 内部服务不可公网访问
- [ ] DDoS 防护已启用
- [ ] Rate limiting 已配置

### 服务器要求
- [ ] CPU: 最少 4 核心，推荐 8 核心

- [ ] 存储: SSD，最少 100GB
- [ ] 网络: 1Gbps 带宽
- [ ] 操作系统: Ubuntu 20.04+ 或 CentOS 8+

### Docker 环境
- [ ] Docker 20.10+ 已安装
- [ ] Docker Compose 2.0+ 已安装
- [ ] Docker 守护进程配置优化
- [ ] 日志驱动配置为 json-file
- [ ] 容器资源限制已设置

### 数据库
